Hash Functions (Hashes)

A hash function definition is that it takes a data of arbitrary value and mapps it to a memory location of fix value.

e.g. ‘Shubham’ and ‘Abhay’ have arbitrary values because they have different amount of alphabets in them (Arbitrary size data). In computer they will be mapped to a location which will be either 16 or 32 or 64 bit memory (Fixed size memory location).

If we can calculate a hash we can directly go to the table where this data is stored. This will help in accessing the large amount of data very fast. That is why dictionary is so fast. It uses hash function.

The hashing function is written in the ‘dictionary’ folder.

The python will randomize the hashes in which the strings are stored. This means that for any character hash value will be the same when the program is running. But, when the program restarts, the hash value will change with every restart (or every time the program is run).

The computer does not understand the lists, tuples, dictionaries. All it knows is memory locations. So all other datatypes are implemented using the linear memory.

There is something called as a ‘hashlib’ in python. This is a library that will provide more information regarding the hash and the security with python.
